ParkHere Site Plan

Bernie Zhong

Overview

Purpose

這個專案的目的是解決在都市中找停車位困難的問題，特別是台灣的大城市如台北和高雄。許多駕駛人浪費大量時間在找車位上，這不僅增加交通壅塞，也造成碳排放增加。我的動機是自己開車時經常遇到這個問題，希望透過這個應用程式，幫助使用者快速找到附近有空位的停車場，提升生活便利性與效率。

Target Audience

駕駛族群，特別是常在都市內開車的人 台灣主要城市（如台北市、高雄市）的居民或訪客 需要臨時找停車場的外地旅客 外送員、物流人員等需要快速停車的工作者

Major Functions

即時位置定位：使用者開啟應用後自動取得目前所在位置，作為搜尋停車場的中心點。 搜尋附近停車場：根據使用者位置或輸入地址，列出周圍的停車場資訊。 顯示剩餘車位數量：即時顯示每個停車場的可用車位（如小客車、大客車、機車）。 地圖標示與導航：整合地圖服務，在地圖上顯示停車場位置，並提供導航連結至 Google Maps 或 Apple Maps。 篩選功能：使用者可依類型（室內/室外）、價格、車種（機車/汽車）等條件進行篩選。 詳細資訊頁面：顯示停車場地址、開放時間、費率資訊、車位類型、是否可預約等。 收藏功能：允許使用者將常用的停車場加入收藏列表，方便快速查找。 更新時間提示：顯示資料的最後更新時間，讓使用者知道資料的即時性與可靠度。

External Data

https://data.taipei

https://data.kcg.gov.tw/

使用的 API 與資料包括： 台北市即時停車資訊（開放資料） 　來源：資料台北（https://data.taipei/） 　用途：查詢即時剩餘車位數、停車場地點與基本資訊。 高雄市智慧停車 API 　來源：高雄市政府資料開放平台（https://data.kcg.gov.tw/） 　用途：提供高雄市停車場即時車位與詳細資訊。 未來如擴充可加入： Google Maps API：地圖與導航功能（不計入兩個主要 API 之內） 需儲存的資料： 使用者收藏清單（本地儲存即可） 最近搜尋紀錄（可選擇性儲存）

Module List

地圖顯示模組：整合地圖 API，顯示停車場位置。 搜尋模組：搜尋附近或特定地址的停車場。 API 資料處理模組：接收並整理台北/高雄開放資料。 列表與詳細頁模組：展示搜尋結果與詳細停車場資訊。 篩選器模組：提供條件篩選（車種、價格等）。 收藏管理模組：增刪使用者收藏的停車場。 導航連結模組：整合第三方導航（如 Google Maps）。 狀態顯示模組：顯示資料更新時間與錯誤訊息。
