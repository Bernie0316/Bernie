ParkHere Site Plan

Bernie Zhong

Overview

Purpose

The main purpose of this project is to help us find the parking place more quickly.

Target Audience

Car driver, especially with those who driving in city.

Major Functions

1.Realtime Position: Get position automatically, and take it as center for parking place searching.
2.Nearby Parking Place Search: Listing information of nearby parking place according user position or enter position.
3.Remain Parking Number: Display the remain parking number of the parking place.
4.Map Display and Navigate: Combine map service. Display the parking place on the map, and provide navigate link to Google Maps or Apple Maps.
5.Filter: Filter with price or car type...etc.
6.Detail Page: Display location, price, remain parking...etc, of the parking place 7.Collect: Allow user Collect Parking Place. 8.Last Modified: Display last modified. Let user know the immediacy, reliability data.

External Data

Taipei city Realtime Parking Data
Source: https://data.taipei/

Gaoxiong City Parking API
Source: https://data.kcg.gov.tw/

Module List

Map Display: Link Map API. Display parking place position.
Search: Search parking place with nearby or specificlly location
API Handling
List and Detail Page
Filter
Collect Manager: Add or delete Parking Place in list.
Navigation link
Status Display: Display modified time and error message.
